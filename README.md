# Discord-Overlay

A Windows tray application that automatically syncs your OBS Discord StreamKit overlay with your current Discord voice channel — switch servers or rooms mid-stream and the overlay follows you, no manual URL editing required.

> **Status:** under active rewrite. The previous DirectX 11 capture-window approach has been retired in favor of the StreamKit URL-sync approach (inspired by [voice-channel-grabber](https://github.com/dichternebel/voice-channel-grabber)) for a friendlier setup. README will be expanded as the new implementation lands.
